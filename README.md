## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/rkymtc/reactnative-app-ui
```

Proje dizinine gidin

```bash
  cd reactnative-app-ui
```

Gerekli paketleri yükleyin

```bash
  npm install
```
```bash
  npm i expo-font
```
```bash
  npm i @expo/vector-icons
```
```bash
  npm i expo-constants
```
```bash
  npm i @mui/material
```

Sunucuyu çalıştırın

```bash
  expo start
```
